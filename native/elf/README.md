# poole-elf

`poole-elf` is the dependency-free `no_std` PKELF1 validator and caller-buffer loader used by the developing PooleBoot boundary.

It accepts only the bounded PooleKernel `ET_DYN` profile documented in `docs/native-elf-loader.md`. The `host-probe` binary exists only for differential qualification and is not part of PooleOS production media.
